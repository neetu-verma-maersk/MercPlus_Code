Pulling criteria for MERCA01, A02 and A03 reports
In each of these reports, the user submits two dates to define a date range. This value is used to test the field RKRP_XMIT_DTE, testing for 1 minute after midnight on the from date through one minute before midnight on the to date.

The STATUS_CODE value must be greater than or equal to 600 and less E than 9000

The RRIS_XMIT_SW must not be "Y"

The user has optional inputs of Shop Vendor and Mode depending on which report is involved. these are, if specified, tested against the

SHOP_CD, MODE, VENDOR_CD fields as applicable.
