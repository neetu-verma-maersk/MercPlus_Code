ASSOCIATE A SPARE PART WITH A MODE/CODE COMBINATION

  • Some STS codes with high value parts (i.e. compressors and controllers) have a list of acceptable parts for that code. This to insure that only the correct parts are used for these repairs.
  • If a vendor enters a valid part (per the MASTER PARTS TABLE) for a code with parts associated to it, and that part is not associated with the code - it will not be accepted.
    • PROCEDURE FOR PART NOT ASSOCIATED WITH CODE under construction

        

TIP - WARNING - If you associate one part with a mode/code combination - you must associate all parts which will be used for the code.

To add new Repair Code/Part Association

 

1) To add new click 'Add New Association'

     

2) Select a manual then mode from dropdowns

3) Enter repair code

4) Enter a part number

5) Click submit

    

 

 

To query and existing Repair Code/Part Association

 

1) Enter manual/mode/code and press query for all parts associated

      

 2) To delete an association – click the delete link at the right of the line.
