MERC+ PTI management
 
PTI  - QUERY
 
  • A P.T.I. (Pre-trip inspection) for reefers must be done before each loaded move.
  • PTI's are good for 60 days
    • PTI default exceptions
      • CA reefers
      • Superfreezers
  • PTI date from RKEM is displayed to vendor and approver when the unit is queried.
    • Since vendor has access to PTI date and is submitting estimates for same - vendor should never PTI a unit which does not need the PTI.

   Setting PTI exception periods

  • CA Reefers and Superfreezers are exceptions to the 60 day PTI rule.
  • Presently exceptions are set at 30 days but this is adjustable
  • Exceptions can be entered by RKEM profile.
