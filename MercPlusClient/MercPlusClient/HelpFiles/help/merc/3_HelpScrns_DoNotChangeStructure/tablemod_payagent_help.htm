MERC+ Pay Agent management
 
Pay Agent.

 The Pay Agent management screen is used to manage the Pay Agent used toward accounting systems RRIS or FACT. The Pay Agent in MERC+ is validated in RRIS, therefore setting up a Pay Agent in MERC+ requires the Pay Agent is existing in RRIS otherwise the Pay Agent will not be accepted in MERC+.
The Pay Agent is only used when automatic transmission to RRIS or FACT takes place, if no automatic transmission of repair costs are to take place it is not necessary to set up a Pay Agent.

Pay Agent, Profit Centers and RRIS format must be obtained from local finance management.
 
Required access levels.

Look up       : CPH or Admin
Create new: CPH or Admin
Edit              : CPH or Admin
Delete         : CPH or Admin

How to create a Pay Agent

1) From MERC+ main screen, click on link named "Pay Agent".

2) Click "ADD NEW" to add a new Pay Agent.
 

3) A blank screen opens

 

4) Insert details and click "Add Record".


The details are now validated in RRIS and MERC+ will revert with result as follow;

Update System message
Successful RRIS Pay Agent Code XXX Added
Pay Agent already exists in MERC+ RRIS Pay Agent Code XXX Already Exists - Not Added
Pay Agent or profit center or combination not found in RRIS RRIS Validation Error(s) have occured Invalid Payagent / Profit Center combination

 

How to edit a Pay Agent

1) From MERC+ main screen, click on link named "Pay Agent".




2) Select the Pay Agent to be edited from drop down.

 

3) Change the details as necessary and Click "Update" to update changes.

How to delete a Pay Agent

1) From MERC+ main screen, click on link named "Pay Agent".


2) Select the Pay Agent to be deleted from drop down.

 

3) Click "Delete" to delete the selected Pay Agent.
