Click on a button in the screen shot to see the features of that button.

         

 

 

 

 

 

         

Edit Records

Clicking this button returns the record to the estimate entry mode where the record can be edited.

 

 

 

 

 

Update Records

This button is clicked to update a record. Data which can be updated on a record are:

 - Date

 - Core number on a submitted estimate

 

 

 

 

Submit Records for approval

Clicking on  button submits record to Maersk Line for approval.
NOTE - estimates that have been submitted can still be edited even after approved by Maersk Line. If estimate is edited after approval - the appoval is rescinded and the  amended estimate must be re-submitted for approval.

 

Delete Records

Clicking this button deletes the record. This step is not reversible and no warning is given after the button is pressed.
