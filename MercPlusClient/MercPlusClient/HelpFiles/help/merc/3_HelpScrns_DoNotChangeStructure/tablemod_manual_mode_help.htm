How to add a new Manual Mode

1) Click ADD NEW


 

 

2) Select a Manual from the dropdown

3) Select a Mode from the dropdown

4) Set the ACTIVE switch to Y or N

5) Click ADD RECORD.

 

Update Manual Mode

1) Select Manual Code from the Drop Down & press 'Update - Get Mode' button
     (not shown)
 


2) Select Mode from the Drop Down & press 'Update - Get Manual Mode' button
3) Update the Manual Mode Active Switch and press 'Update'
4) Click update –get mode

      

 

5) Click update get manual mode
