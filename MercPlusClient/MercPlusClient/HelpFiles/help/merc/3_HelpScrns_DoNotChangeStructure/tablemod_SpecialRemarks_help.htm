To add new remarks

1) Click the ADD NEW REMARKS button. This will cause bottom half of screen to appear.
2) Select the criteria to be used - either:
            An RKEM profile
                or
            An equipment number range
3) Select the REPAIR CEILING (see chart above for reactions to various amounts)
4) Select DISPLAY REMARKS TO VENDOR - Y/N
         Y - remarks will appear on vendors screen when he ????????  
         N - remarks will appear only to MSL.
5) Enter the desired remarks.

Special Remarks - Number Range
The number range is inclusive. Any range within a range will override the remarks of the original range.

Example

 
From To Remarks
MAEU5550001 MAEU5559991 Check Fan

   

Any estimate entered for equipment in the above range will display "CHECK FAN" to vendor.
However, if we put in another range:
From To Remarks
MAEU5550501 MAEU5550601 Check compressor for leaks

Note that for the above 10 units - only the compressor message will display. This can be overcome by making the message for the 10 units " check fan and compressor for leaks".
