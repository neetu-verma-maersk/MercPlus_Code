Edit an existing user

Editing an existing user may be necessary from time to another.

TIP
In case a user has had a new certificate issued and need the same rights in system as with the old certificate all you need to do is to change the Certificate Key to the new Certificate number.

How to edit an existing user

1) Select the user's country from the dropdown

 

  • TIP; to go to e.g. "U" for USA press letter u on keyboard. Every time you click u on keyboard next country starting with U will appear. When correct country is highlighted hit enter on keyboard.

2) Select user for update from dropdown

 

  • When a country has been chosen, all users in selected country will then be available in the User dropdown.
    Chose the user to be edited and Click "Edit".

3) The users information will appear and can be edited.

 

  • Change the parameters

4) Edit changes

 

  • Press "Update" if changes are to be kept. Press the back button if they are not to be kept

5) Confirm update of user

 

  • Press "Confirm Update" to finally update user data

 

6) System confirmation of user update.
