How to add a Repair STS Code

1) Select  Manual Code from the Drop Down
     
    


2) Select  Mode from Drop Down

    

 

3) Select an Index and click 'Go To Add Repair Code' button

   

 

4) Fill in the records

   

 

5) Click Add New button

Update / Delete repair codes

1) Select  Manual Code from the Drop Down

    


Step 2 -  Select  Mode from the Drop Down

3)  Select  Index & Select Repair Code from the Drop Down and click Select Repair Code

     

4) Make changes and press 'Update' Button OR press 'Delete' to delete the record
