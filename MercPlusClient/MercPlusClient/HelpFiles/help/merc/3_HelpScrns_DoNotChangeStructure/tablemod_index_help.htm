To add an Index

1)  Click the  'Add - Get Mode' button

     

 

2) Select a Manual Code (usually MAER)

3) Click the 'Add - Get Mode' button

     

3) Select a Mode (not shown)

4) Click 'AddNew' button (not shown)

5) Add index ID and Index Description

6) Click 'Add Record' button

Update / Delete Index

1) Select Manual Code from the Drop Down & press 'Update - Get Mode' button
    

2)  Select Mode from the Drop Down & press 'Update - Get Index' button
    

 

3) Select Index from the Drop Down & press 'Update - Get Index Record' button (not shown)
 

4) Make Changes & press 'Update' button OR press 'Delete' to delete the record
