MERC+ Exclusive Repair Code management
 

EXCLUSIONARY CODES

Codes which are exclusionary cannot co-exist on the same estimate.
For example code to replace a component and code to repair same component would be exclusionary.

RULES FOR ADDING EXCLUSIONARY PAIRS

  • When adding exclusionary pairs - it is only necessary to add the pair one time. Example add code 1112 then 1113 - you are not able to add 1113 then 1112.
  • Always start with the lowest code and add the highest code 2nd.

How to add / delete  repair code exclusive
 

1) Select  Manual Code from the Drop Down & press 'Select Mode' button

2)  Select  Mode from the Drop Down & press 'Select Repair Code' button


Step 3 -  Select Lower of the 2 Repair Codes from the drop down & press 'Go to Add Delete' button



Step 4 -  Provide Exclusive Repair Code to add a new Exclusive OR press 'Delete Record' to delete existing repair code exclusives
