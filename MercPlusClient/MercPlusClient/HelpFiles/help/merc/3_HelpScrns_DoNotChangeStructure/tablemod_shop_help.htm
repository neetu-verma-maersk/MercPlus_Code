Shop Profile
 

Related Topic - Different Shop Types

OVERVIEW

This screen is used to allow the users to create and manage repair shops and to control various approval levels and settings for shops. Further it provides users with an interface where users can Query, Update/Revise and Add new Shops.

AUTHORITY

  • CREATE/EDIT
    Only System Administrator and CPH have access to Create a new shop and only these two user types are able to Edit vendor code and GEO location for an existing shop.

    Area can, when a shop has been created, activate and deactivate a shop and edit all fields, except the above mentioned.

    It is not necessary to deactivate a shop in order to change settings for System Administrator, CPH and Area users.

     Note: When a shop has been created it is not possible for any user to change the shop code, nor is it possible to delete a shop.
     

  • READ/EDIT
    Country, Region, MSL (Location/Local EMR) have access to View and Edit all other fields in shop profile provided the shop is set to inactive.

QUERY 

For querying Shops; users will be able to select and query for an Existing Shops from the Drop Down list. Once the User presses the ‘Query’ button the rest of the fields (information) for the selected shop code will be populated.
 Note: Shops available in the Drop Down options will be based on the User Permissions

CREATE / UPDATE / DELETE 

Users will be able to view and/or edit the Shop Profile from this screen. Following are the fields available for update:

  1. Shop code

  2. Shop Description

  3. Shop Type (1,2,3 or 4)

  4. Vendor Code (editable only by System Administrator or CPH users)

  5. RKRP Location (Text or Display)

  6. Geo Location (editable only by System Administrator or CPH users)

  7. Currency Code

  8. Email Address

  9. Phone

  10. Fax

  11. RRIS Suffix Code

  12. Discount/Mark up

  13. Shop Active Switch Y/N (editable only by System Administrator, CPH and Area users)

  14. Preparation Time Y/N - Default Y

  15. RRIS Transmit Switch Y/N (auto payment)

  16. OT Suspend switch Y/N

  17. Container Sales Tax Parts

  18. Container Sales Tax Labor

  19. Chassis/Genset Sales Tax Parts

  20. Chassis/Genset Sales Tax Labor

  21. Import Tax Parts

  22. ACEP switch Y/N

Note: When a new shop has been created by CPH – Only the fields in Bold is populated. Area, Country, Region or MSL user will then  provide the rest of the information.

ADD NEW

CPH and ADMIN will be able to add a new Shop record by clicking on the ‘Add New Shop’ button provided on the Search Screen. CPH and ADMIN will be able to add a new Shop by providing just the following information:

  • Shop Code

  • Geo Location

  • Shop Active Switch Y/N

 

Note: In Order for Users to Update / Populate the editable fields – Admin, CPH or Area should set the Shop Active Switch as ‘N’

Taxes
If the Tax fields in Shop Table are = 0 (Zero) – Users will be given the ability to provide Tax Amounts from the estimate Entry Screen.
If the Tax fields in Shop Table are populated i.e.  Not = 0 (Zero) – Users will not be given the ability to provide Tax Amounts from the estimate Entry Screen but the System will calculate the taxes wherever applicable.
