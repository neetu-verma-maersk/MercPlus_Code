OVERVIEW

The screen is used to query the following for Estimates.

  • Average Time to Estimate - Average number of days from 'GID (gate in date)' date to 'Estimate' date for all estimates under selected criteria.
  • Average Time to Authorise - Average number of days from 'Estimate' date to 'Authorized' date for all approved estimates under selected criteria.
  • Average Time to Repair - Average number of days from 'Authorized' date to 'Completion' date for all completed estimates under selected criteria.

How to Query at different levels for repair time productivity (not shop user)

1) Country is selected by default.

2) Give the 'From Date', 'To Date' , select the country from dropdown menu and click 'Query'.

3) To query at location level, select 'Location' radio button, fill in the date range, select the location from dropdown menu and click 'Query'.

4) To query at Shop level, select 'Shop' radio button, fill in the date range, select the shop from dropdown menu and click 'Query'.
