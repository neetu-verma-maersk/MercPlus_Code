Manage Vendor Tables
 

OVERVIEW

This screen will be used to manage Vendors. It provides users with an interface where they update existing Vendors and add new Vendors.

AUTHORITY 

Only Administrator (System Admin, Prime Mover), CPH, Area, Country, Region, MSL (Location/Local EMR) will have access to the Vendor Screens.

UPDATE  

Users will be able to select and look at the details of all the existing Vendors. Upon selecting the Vendor Code from the Drop Down (All existing vendors in the Vendor Table) the rest of the fields will be populated i.e. Vendor Description, Vendor Active Switch and Change User information.

From this Screen users will be able to update existing Vendors.
