MERC+ Prep Time management
 
Prep Time

Preptime is added to an estimate in 2 ways.

  • Automatically
    • Automatic update of  prep time takes place when an estimate is entered via the internet
    • Automatic update of  prep time takes place if estimate has been submitted via the generic interface without preptime codes added.
  • Manually
    • If a shop transmits an estimate via the generic interface with preptime codes added, MERC recognizes the prep time codes and validates them.
    • If a shop attempts to enter the prep time codes via browser, they will not be accepted.

To add Prep Time
   
1) Insure the prep time code is in the manual (as an ordinary STS code)

2) Click 'ADD NEW' button (not shown)

2) Select the MODE from the dropdown

3) Enter the PREP TIME MAXIMUM (normally <=2,8,24,40 and >40)

4) Enter the PREP CODE (see step 1)

5) Enter the PREP HOURS - the prep time that will be given.

6) Click the 'Add Record' button.

   

 

 

 

 Update / Delete existing Prep Time Record

1) Select Mode from the Drop Down & press 'Update - Get Preptime Max' button

       

 
2) Select Prep Time Max for the Mode & press ' Update - Get Prep Time' button
      

 
3) Make changes and press 'Update' button OR press 'Delete' button to delete.
