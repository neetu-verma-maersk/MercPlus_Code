Repair Location Code

 

OVERVIEW

The container location coding convention locates damages within an area as large as a complete face of the container or as small as a nominal 1200 mm x 1200 mm (4 ft x 4 ft) square, or even less for the main components located on its edges (rails, corner posts).

The location code shall consist of four characters depending upon the area to be described. It identifies the smallest area containing the entire vertical and horizontal length of the damage.

This screen provides an interface, where user can view the first two characters of repair location codes. The last two characters are accepted in Merc+ as free text.

How to Query an existing Repair Location Code

1) Select a Repair Location Code from dropdown and click on 'Query'. All data will be returned as indicated.
