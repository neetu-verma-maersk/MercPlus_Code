MERC+ User management
 
Add a new user
A new user is updated in MERC+ in 2 steps
  1. Create the new user in "Add new user" screen.
  2. Assign the user permissions in "Set user permissions" screen.

NOTE
Adding a new user to MERC+ without assigning user permissions will not allow user to log on to MERC+.

1) How to Add a New User

On MERC+ main screen click on "User Management" in lower right corner.
A new screen will appear. Click on "Add new user".
Below screen will now appear. Fill out requested data.
All fields must be filled out

 

  • Certificate Key
    The Certificate Key is the number returned from CAF when a user applies for a certificate.
    In CAF this is known as MERC+ Internal or MERC+ External ID. You must bring this ID from CAF in order to create a new user in MERC.
     
  • First name
    First name of the user.
     
  • Last name
    Last name of the user.
    NOTE; In order to easily find Internal users in MERC+ user management, you MUST insert the internal users @maersk ID before the users last name with a space between @maersk ID and last name (see above).
    For External users you must write EXT before the users last name with a space between EXT and last name
    The above will greatly help find users if you later on are going to Edit them.
     
  • Company name
    Company name of the user. Correct company name must always be updated for internal as well as external users. It is appreciated full company name is used also for internal users. Do not write e.g. Maersk or MSL but Maersk Inc or Maersk India Pvt Ltd
     
  • Location code
    The Location Code is the Geo code for the users location (New Orleans USA = USNOL)
     
  • Approval amount.
    For users with MSL, REGION, COUNTRY, AREA and CPH access only
    Users with "SHOP" and  "VENDOR" access are always set to 0.
    The approval amount is the amount for a single estimate that the user can approve. The amount is decided by his manager or accounting.
    Setting this field to a value of USD 500 will thus allow the user to approve estimates up to USD 500. Estimates in excess of USD 500 will have to be approved by a user with higher approval limit.
    Since MERC is linked to RRIS, approval of an estimate will thus also approve payment and is therefore of UTMOST importance the approval amount is set according to input from the users manager or accounting. It is not up to the MERC+ Super user to determine this amount.
     
  • Active
    If the active switch is set to N - the user will not be able to login to MERC.

When all details of new user has been entered click on "Add".

2) Confirm new user

 

A new screen will open asking you to confirm what you have just entered is correct
Click "Confirm" if all details are correct.
Hit backspace on keyboard to go back and change details.
