Manage Country Contract Screen
 

OVERVIEW    

Note :- Country contracts can neither be updated nor added effective from 22nd November 2009

This screen is used to manage contracts for material cost pr repair code between a country and CPH, i.e. the agreement for decentralized countries. It provides users with an interface where users can Query, Update, Delete and Add new Contracts.

The country contract is to be used only for decentralized countries where CPH has the contract with the agency (as opposed to centralized countries where CPH has the contract directly with a shop).

Country contract is NOT to be used for numbered spare parts (reefer/genset parts)

While the shop contract ensures that the shop cannot charge higher material cost for a code than agreed, the country contract ensures that CPH cannot be charged higher material code for a repair code than agreed with the agency/country.
So; the shop contract for each shop stipulates what the agency max pays their shop for a code while the country contract stipulates what CPH max pays pays the agency. From above it is obvious that there will by as many shop contracts as there are shops in a country but only 1 country contract.
When the shop is creating an estimate the material cost entered for each code will be checked against the shop contract, and if exceeding the agreement, a warning will be issued telling the shop they have exceeded the agreed material amount for the code.

When the estimate is submitted for approval at local MSL, MERC+ will convert the amount for each code to agency/CPH agreement.

Who can see what?

  • SHOP;
    Estimate will reflect the agreement between shop and agency, i.e. the material cost pr code will be shown on "shop level".

  • AGENCY;
    Estimate will reflect the agreement between shop and agency AND the agreement between agency and CPH, i.e. the material cost pr code will be shown on "shop level" AND on "CPH level".

  • CPH;
    Estimate will reflect the agreement between agency and CPH, i.e. the material cost pr code will be shown on "CPH level".

It is under no circumstances possible for a shop to see the agreement between agency and CPH nor is it possible to pull reports showing this.

It is under no circumstances possible for CPH to see the agreement between agency and shop nor is it possible to pull reports showing this.

A few guidelines and rules

  • Cost for "numbered" spare parts, i.e. spare parts for reefers and gensets are controlled by entering the part number directly on the estimate, MERC+ will then calculate the correct price based on the discount/mark up agreement with shop for reefer/genset parts. Under no circumstances shall the contract contain part costs. However any additional "material" cost (ancillaries like solder, gas etc) which would be necessary to replace e.g. a condenser coil shall be included in the contract in order to ensure such costs not are abused.

  • Each shop must have a shop contract updated in MERC+, the country contract will not be in force if there are no shop contract(s).

  • If a material cost agreement for a code cannot be found on the country contract, MERC+ will reuse the material cost for the code entered by the shop.

 

AUTHORITY  

Only System Administrator, CPH, AREA and Country will have access the Country Contract Screens.

QUERY 

For querying Country contract Screen 'Country’ is Mandatory - Also search can be further filtered by Repair Code and/or mode.

The result set will be sorted in the following Order:

  • 1 Country

  • 2 Repair Code

  • 3 Manual Code

  • 4 Mode

  • 5 Eff Date

The result set will display the following:

  •  Country

  •  Repair Code

  •  Amount + (Country Currency)

  •  Manual

  •  Mode

  •  Effective Date

  •  Exp Date

Note: The Countries in the Drop Down will be all existing Countries and Modes in the Mode Drop Down will be all existing Modes (Active & Inactive)
