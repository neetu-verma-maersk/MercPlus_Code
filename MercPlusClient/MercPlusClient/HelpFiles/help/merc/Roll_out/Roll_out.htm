MERC+ roll out
 

Roll out of MERC+ has been completed world wide July 1st 2005.

 

Roll out plan has been laid down following the below criteria's.

  1. WMNR countries have first priority in order to decommission WMNR October 1st 2004.
    (WMNR has now been decommissioned)
  2. Big/major repair locations have second priority.
  3. The US will roll out themselves assisted by GLBEMR.
  4. Last locations to start using MERC+ will be small/minor repair shops.

Prior rollout in a shop can start,  the following must be set up.

- MSL users approving work orders for the shop must apply for a certificate and be granted access to MERC+
- Shop users that needs to update estimates must apply for a certificate and be granted access to MERC+

MERC+ must, in advance of intended roll out date, as a minimum be set up with

  1. Shop labour rates (centralized and decentralized locations/countries).
  2. Shop material rates (centralized and decentralized locations/countries.
  3. Country labour rates (decentralized countries only)
  4. Country material rates (decentralized countries only)
  5. When above is finished the shop active switch is set to "Y" and then
  6. Customer/shop/mode association is set up (if automatic RRIS update not used you must use Payagent code FMU).
