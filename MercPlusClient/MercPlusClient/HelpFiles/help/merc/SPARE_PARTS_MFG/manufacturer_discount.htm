MANUFACTURER DISCOUNT

  • The manufacturer's discount covers the discount on reefer and genset parts by manufacturer.
    • It is not possible to set separate discounts for gensets vs. reefers.
  • The manufacturer's discount is set by CPH.
  • The manufacturer's discount is given to Type 1 shops
  • The manufacturer's discount will generally be altered yearly.

To add record - click ADD NEW MANUFACTURER

Enter the Manufacturer Code, Name and Discount Percentage.
 

 

To Query existing – select from dropdown and press query
