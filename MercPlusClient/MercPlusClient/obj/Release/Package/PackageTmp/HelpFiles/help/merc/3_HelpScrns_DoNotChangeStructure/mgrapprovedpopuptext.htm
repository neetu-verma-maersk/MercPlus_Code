Estimate

  • Edit Records - Clicking this button returns the record to the estimate entry mode where the record can be edited.
     

  • Update Records - This button is clicked to update a record. Data which can be updated on a record are:
    - Date
    - Core number on a submitted estimate
     
  • Submit Records for approval - Clicking on  button submits record to Maersk Line for approval.

    NOTE - estimates that have been submitted can still be edited even after approved by Maersk Line. If estimate is edited after approval

- the appoval is rescinded and the  amended estimate must be re-submitted for approval.
 

  • Delete Records - Clicking this button deletes the record. This step is not reversible and no warning is given after the button is pressed.
     
  • <<    Navigation button (previous record) - Goes to the previous record in the query
     
  • <<< Navigation button (first record) - Goes to the first record in the query
     
  • New Query - Returns to the ESTIMATE QUERY SCREEN
     
  • >>    Navigation button (next record) - Goes to the next record in the query
     
  • >>> Navigation button (last record) - Goes to the last record of the query.
     
  • Print - Brings up the PRINT window
