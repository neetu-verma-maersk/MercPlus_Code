MODIFY AUTH GROUP

Strongly recommended this screen is not to be changed without extensive research and consultation with application support team.

Associate user with auth group

Step 1 - Select Country from the Drop Down to filter the Users
Step 2 - Select User from the Drop Down and press 'Manage AuthGroup' Button



Step 3 - Select Authgroup from the Drop Down and Press 'Associate User with AuthGroup' Button

 

Step 4 -  Select from Available list and click '>' or '>>' Button to Add an Auth Group
                    OR '<' or '<<' to remove existing Auth Group(s)
Step 5 - Press 'Associate User with AuthGroup' Button

 

modify auth group permissions

Step 1 - Select Country from the Drop Down to filter the Users
Step 2 - Select User from the Drop Down and press 'Manage AuthGroup' Button

 

Step 3 - To remove All User - AuthGroup Permissions press 'Remove User - AuthGroup Permissions' Button
              To modify AuthGroup Permissions press 'Modify AuthGroup Permissions' Button

 

Step 4 -  Update the Authgroup Website Permissions (Check Boxes) & press  'Update Website Access' Button

 

Step 5 -  Click on the Edit Permissions for Website Tab to Edit permissions for the
                     website & press 'Update Website Access' Button
