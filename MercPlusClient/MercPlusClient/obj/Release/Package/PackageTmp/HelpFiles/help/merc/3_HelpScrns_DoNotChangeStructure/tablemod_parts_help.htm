How to query a part

1) Fill in criteria and press QUERY

Tip - A part can be searched by:
          - Part Group
          - MFG Code
          - Designation
          - Active Y/N
          - Core Y/N
          - Deduct Core Value Y/N
          - Keyword/phrase *

* Entering COMPESSOR will return any and all parts with the words compressor or compressors. Entering ELE will return ELEment, ELEctrical, ELEctronic etc.
