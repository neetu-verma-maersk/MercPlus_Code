   Problems pulling reports? - Click here

  • Date From –  First day to be included in your report, please use date format yyyy-mm-dd
  • Date To – Last day to be included in your report, please use date format yyyy-mm-dd
  • Shop – choose the shop which you want to pull the report of
  • Customer – please select ‘MAER’ in the drop down menu.
  • Mode – type of the container, e.g. 43 Reefer unit, 03 Steel Box etc
  • Equipment Number – Please enter the full container number e.g. MAEU5720000
  • Repair Code – code from the STS Manual categorizing the type of the repair needed on the container.
  • Reports - here listed the types of the reports available within your authorization limits
  • Report Instructions - give the details step by step of how to pull a report of respective type. Please read carefully before you pull the report.
