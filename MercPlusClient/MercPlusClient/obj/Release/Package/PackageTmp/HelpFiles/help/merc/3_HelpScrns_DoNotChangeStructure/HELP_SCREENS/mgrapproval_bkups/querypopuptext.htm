QUERY HEADER

  • Shop Code – the 3 Character code (number) assigned to your shop in MERC+. If set to ALL, the query will give you result for all your shops.


     

  • Customer – please select ‘MAER’ in the drop down menu.


     
  • Mode – type of the container, e.g. 43 Reefer unit, 03 Steel Box etc. If set to ALL, the query will give you result for all modes.


     
  • Equipment Number – The number of the equipment on which you want to do a query. Prefix and all digits (including check digit) without spaces must be entered.


     
  • Vendor Ref (optional) – If you want to look up a specific estimate or work order you should use this field. Since the vendor reference number is unique it will always only give you 1 result.


     
  • Location - the location where the repair activities take place. If set to ALL, the query will give you result for all locations where you have shops.


     

  • Query Type - in addition to the daily query, there are other querying options available for tracking estimates of different types and status:

    • Daily - is the default query on the query screen. The daily query can be run by clicking the query button. It returns all estimates that are active, including:
      • Approved estimates that have not been assigned to be worked.
      • Rejected estimates that have not be corrected or deleted.
      • Drafts of estimates (estimates that have not been submitted).

The daily query return estimates for all shops under the vendor's control. If desired, the daily query can be filtered by selecting a shop from the dropdown before running the query.

  • Working - Estimates that have been assigned to a mechanic
  • Rejected - Estimates that have been rejected by Maersk Line
  • Under Review
  • Approved - Estimates that have been approved but not assigned
  • Completed
  • RRIS Transmitted
  • RRIS Rejected
  • RRIS Accepted
  • Processed

     
  • Sort by Amount – You are able to sort your query by total cost of repair. This can be accomplished by putting a check in the SORT BY AMOUNT box


     
  • Query - When all criteria have been selected - simply click on the QUERY button.
