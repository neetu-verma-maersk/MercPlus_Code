OVERVIEW

This screen is used to allow the user to retransmit Work Orders to Fact/RRIS. It gives the user the ability to add a few work orders in the screen or upload work orders from an excel file in case of bulk work orders to be re-transmitted. The


AUTHORITY

Only ADMIN can re-transmit work orders.

 

How to re-transmit work orders by giving the work order I.D.'s manually through the screen

1) Enter a work order id in the text box shown and click 'Add >>"

 

2) After adding workorders, click on "Process WorkOrder

This will process the work orders, and give an error/success message on the screen at the bottom.

How to re-transmit work orders through Excel file

1) Click on "Browse", select the file with work order I.D.'s to be retransmitted.

Rules for creating the excel:-

  • Enter Work order I.D.'s in the first sheet
  • Work order I.D.'s shoud be in the 1st column from 2nd row
  • There should be no blank row bwtween work orders

 

2) Click on Process file. This will process the work order I.D.'s and give the relevant error/success message.

 

3) On clicking "Process file", you may get the following dialog box. Select "Yes".
