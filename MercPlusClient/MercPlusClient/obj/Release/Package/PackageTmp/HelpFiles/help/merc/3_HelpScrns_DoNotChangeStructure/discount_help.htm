Shop Profile - Discount/Markup (shop type 3)
 
Manufacturer Discount/Markup

 

Related Links

  • How to set up discount/markup for individual manufacturers
  • How to set up a discount/markup for all manufacturers other than the ones specified in the dropdowns

OVERVIEW

This screen is used to allow the user who have shop type 3, to have more than one manufacturers discount set up in the shop profile screen. The functionality is such that if no values are selected in the drop down boxes then it would apply the value of the field "Discount/Markup - All Manufacturers" to the parts used in the work orders. If there is a manufacturer selected and a corresponding value then it is this value which is used against the work order


AUTHORITY

Users with Shop type 3.

 

How to set up discount/markup for individual manufacturers  

1) Select a 'Manufacturer code' from dropdown (Upto 10 manufacturer codes can be selected through all the dropdowns) .

2) Enter a value for manufacturer 'Discount/Markup' .

3) Click Submit . A message will be displayed as shown .

A total of 10 manufacturer dicount/markup can be specified in this way

How to set up a discount/markup for all manufacturers other than the ones specified in the dropdowns

1) Enter a value for 'Discount/Markup - All Manufacturers'.

1) Click Submit . A message will be displayed as shown .

This value will be applied to the parts used in work orders for manufacturers not selected in any of the dropdowns .
