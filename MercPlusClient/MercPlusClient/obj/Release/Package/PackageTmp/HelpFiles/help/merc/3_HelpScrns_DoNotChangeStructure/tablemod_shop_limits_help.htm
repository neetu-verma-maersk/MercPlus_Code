Manage Shop Limits
 

OVERVIEW

This screen will be used to create and manage auto-approve and suspend limits for shop/mode pairs. Shop Limits Screens provide users with an interface where they can update and add Shop Limits. Users can search for existing Limits by Shop/Mode.

AUTHORITY

Only Administrator (System Admin, Prime Mover), CPH, Area, Country, Region, MSL (Location/ Local EMR) has access to the Shop Limit Screens.

QUERY / UPDATE 
 

Users will first have to select the Shop Code from the Drop Down List and press ‘Update - Select Mode’ button to get all the existing Modes for the selected Shop in Shop Limits Table.
Note: Only existing Shops in Shop Limits Table will be available for selection and Users will only be able to see Shops under their Permission Levels

      


 

 

 


After selecting the Mode from the Drop Down List, user will have to press ‘Update – Shop Limits’ button to go
to the update Screen




 

 

 


From this Screen users will be able to Update the selected Shop Limits record.

ADD NEW

Users can Add new Shop Limits by clicking on the Add New Button. From this Screen users will be able to Add a new Shop Limits for Shops under their Permissions Level.

Users will be able to add a new Shop Limit by providing the following information:

  •  Shop Code

  •  Shop Mode

  •  Suspend (Repair Amount) limit

  •  Shop Materials Limit

Note: All Shop Codes (Active and Inactive) based on permissions from Shop Table and Active Modes from Mode Table will be available for selection
