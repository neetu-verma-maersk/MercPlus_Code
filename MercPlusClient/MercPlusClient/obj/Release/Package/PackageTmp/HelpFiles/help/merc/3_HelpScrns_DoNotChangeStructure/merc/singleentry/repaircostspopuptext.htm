REPAIR COST

  • Repair Code – the code from the STS Manual categorizing the type of the container repair. A click on + will expand the available repair fields from 20 to 45.
  • Pcs – the number of times that repair code will be used on work order.
  • Material Cost per Piece – the cost of the material needed for this repair activity as per repair contract.
  • Man Hours per Piece – the number of the working hours needed for the repair activity on each piece. MERC+ will only accept up to the maximum hours per code, as stated in the STS manual.
