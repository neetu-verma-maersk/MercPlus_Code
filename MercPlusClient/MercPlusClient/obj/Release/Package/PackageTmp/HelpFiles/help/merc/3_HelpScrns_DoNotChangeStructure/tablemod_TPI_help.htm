OVERVIEW

This screen is used to allow the user to add, modify and delete Third Party Indicators. It provides an interface, where user can view TPI details.


AUTHORITY

ADMIN can update / delete a TPI. Other users can only query them.

 

How to query an existing Third Party Indicator (CPH/Area/Country/Region/MSL/Vendor)

1) Select a TPI code from dropdown and click on 'Query'. All data will be returned as indicated.

 

 

How to Query / Edit / Delete an existing Third Party Indicator - (Admin)

1) Select a TPI code from dropdown and click on 'Query'. All data will be returned as indicated.

Make changes and click submit to update.
Click Delete to remove the TPI code.

How to Add a new TPI Third Party Indicator - (Admin)

1) Click on "Add New TPI Code".

2) Fill in all the details as shown.    

  • 'Third Party Indicator' will be one characters long.
  • The 'Name' and 'Description' fields cannot be left blank.
  • 'Numerical code' has be 5 digit long.
  • 'Category' has to be selected from the drop down. It can be left blank as well.

3) Click Submit. A message will be displayed as shown
