COUNTRY TABLE & CPH LIMIT ADJUSTMENT FACTOR

TIP;

  • The country factor for the CPH level is used to adjust the CPH level to suit conditions in each country.
  • The country factor is set by CPH
  • Example
    • 20 steel box, 2 years old with CPH limit of 1000.00 would be modified as follows
      • Located in United States (country factor set to 75%) = 750.00 limit
      • Located in China (country factor set to 120%) = 1200.00

How to edit the CPH LIMIT ADJUSTMENT FACTOR

1) Pick the country from the dropdown list. 

  • TIP; Records to the COUNTRY screen cannot be added. In addition, the only field which can be edited is the CPH Limit Adjustment Factor.


 

2) - Edit the CPH Limit Adjustment Factor as desired and press SUBMIT.
