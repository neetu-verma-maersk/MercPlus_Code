Manage Customer Shop Mode Screen
 

OVERVIEW

This screen will be used to manage the Customer/Shop/Mode/Payagent Association. It provides users with an interface where users can Query, Update, Delete and Add new Customer/Shop/Mode/Payagent Association.

AUTHORITY 

Only Administrator (System Admin, Prime Mover), CPH, Area, Country, Region, MSL (Location/Local EMR) will have access to the Equipment Type Mode Screens.

QUERY 

Users will be able to narrow their search down by Customer – Required, Shop – Required, and Mode – Optional.  The result set will display the following:

  • Customer

  • Shop

  • Mode

  • Pay agent

  • Corp Payagent

  • RRIS Format

  • Corp Profit Center

  • Profit Center

  • Expense Code

Results will be sorted by Customer, Shop, and Mode - In that Order

 
UPDATE

Users will be able to view and or edit Customer/Shop/Mode/PayAgent Association by clicking on the edit link next to the record. Following fields are editable:

  • Pay agent code

  • Corporate Profit center

  • Corporate PayAgent

  • Profit center

  • Expense code  (Account Code)

  • RRIS Format
