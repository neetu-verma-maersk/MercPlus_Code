HEADER

  • Shop Code – the only code recognized by the system, which stands for your repair shop.

  • Customer – please select ‘MAER’ in the drop down menu.
  • Cause – meaning the cause of the damage, e.g. Wear & Tear.
  • 3rd party port – if it is the 3rd party damage, please fill in the location code of the 3rd party responsible.
  • Currency – system will give the default currency according to the respective repair contract.
  • Equipment Number – container number.
  • Vendor Ref (optional) – This field has space for 10 characters which can be filled in by the vendor, or it can be left blank. If the field is left blank, MERC+ will generate one on its own. The number will be used as reference in future payments.
  • Mode – type of the container, e.g. 43 Reefer unit, 03 Steel Box etc
  • Remarks – all information you want to share with local Maersk office, including general conditions of the container, description of the main damage, suggestion of the repair etc.
  • Request MSL Agency Comments – please tick if comments of local Maersk office are necessary.
  • Equipment Information – system automatically reveals the container profile after entering the Equipment Number.
      • Type – container type, e.g. CONT/REEF
      • Profile  – gives the information of the series the container belongs to
      • Size – container size, e.g. 40/9’6
      • Material – the material the container made of, e.g. SST (stainless steel)
      • Box MFG –manufacturer of the container
      • In Service – when the container start the service
      • Reefer make/model – the manufacturer of the reefer unit and the model information
      • PTI Date – the latest Pre Trip Inspection date, if available
      • Genset make/model

SUBMITTING ESTIMATES

  • HOW TO SUBMIT - When an estimate is complete - it can be submitted for approval by the vendor. Submission is simply a matter of pressing the SUBMIT button on the top or bottom of the estimate creation screen.
  • IF YOU MAKE A MISTAKE AFTER SUBMISSION - After an estimate has been submitted, it can still be changed by the shop/vendor. Any estimate can be called back at any time via the query screen. You will have to search for submitted estimates on query screen.  When an estimate is submitted as completed it is however no longer possible to change anything, not even by system administrator.

REVIEW ESTIMATES   

    • REVIEW ESTIMATES - estimates can be reviewed during the creation to inspect any error inputs. System will indicate what the error is.

ESTIMATE DRAFTS - Estimates can be started and saved as they are not ready to be submitted. This might occur due to a number of reasons, such as holding the box repair estimate for a reefer until the unit repair estimate is ready.

    • SAVING DRAFTS - Estimates can be saved as drafts by clicking the button on the top and bottom of the estimate creation screen.
    • ALTERING DRAFTS - While they are being held as drafts, estimates can be edited, added to or items taken away - and re-saved.
    • DELETING DRAFTS - If it is decided to not submit a draft, the draft can be deleted if desired by going the individual draft and clicking the clear button.

      NOTE : both functions of Altering Drafts and Deleting Drafts can be available after activating the Daily Query.

GET HOURS

    • Get Hours – click the button, system will advise the maximum man hours allowed for the specific repair code.

CLEAR

    • Clear – clear all the input or content on the estimate, even the information previously saved.

PRINT

    • Print – print the current estimate
