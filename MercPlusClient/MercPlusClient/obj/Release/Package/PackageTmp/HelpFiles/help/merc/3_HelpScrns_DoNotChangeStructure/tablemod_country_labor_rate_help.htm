MERC+ Country Labour Rate management
 
COUNTRY TO CPH LABOR RATES

The rates set between decentralized countries and Copenhagen are input by CPH into MERC+.


TIP  If there are no rates set between a country and CPH, the rates of the vendor will pass straight through to CPH.
