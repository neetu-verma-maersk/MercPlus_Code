Manage Shop Labor Rates
 

OVERVIEW

This screen will be used to create and manage Labor Rates for each Shop. It provides users with an interface where users can Query, Update and Add new Labor Rates.

AUTHORITY 

Only Country, Region, MSL (Location/Local EMR) will have access to the Shop Contract Screens.

QUERY 

For querying Shop Labor Rates 'Shop’ is Mandatory (Cannot do a search without selecting a Shop) - Also search can be further filtered by Customer and Equipment Type in order to display all the existing Labor rates for the selected Shop.

Note: Users will only see Labor rates for Shops under their permission levels


UPDATE  

Note: Users will not be able to update Shop Labor Rates when shop active switch = 'y' – Users will have to request CPH/Admin to make the shop inactive in order to edit the Labor Rates.

Users will be able to view and or edit Shop Labor Rates by clicking on the edit link next to the record. Will only be able to edit the following information:

  •  Effective date

  •  Expiration date

  •  Regular Rate

  •  O/T rate

  •  Double time rate

  •  Miscellaneous rate

Note: If Exp date less than current date no edit link will be available

Note: The new Effective Date (If a Labor Rate already exists) must be one day after the Expiration Date       of the Existing Labor Rate

Note: Expiration date must be greater than or equal to the effective date.

Note: Expiration date must not be less than Current Date

Note: Date format expected as YYYY-MM-DD

ADD NEW 

Note: Users will be able to add Labor Rates only when shop active switch = ‘N’ - Users will have to request CPH/Admin to make the shop inactive in order to add a new Labor Rate for a Shop.

Users will be able to add new Shop Labor Rates by clicking on the ‘Add New Rates’ button provided on the Search Screen which will take them to an “Add New Page’.

Note: If Exp date less than current date no edit link will be available
