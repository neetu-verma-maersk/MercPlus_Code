Shop Active Switch - how does it work
When Shop Active is set to 'Y', no changes can be made to the shop rates or contracts - only CPH can make changes to Shop Details.
When Shop Active is set to 'N', then the details, rates and contract can be altered.
 

 

 

 

 

How to Check a email interface text message for errors

To test an email message for errors (80 chars per line etc.) download tester.xls and copy the email in it.
