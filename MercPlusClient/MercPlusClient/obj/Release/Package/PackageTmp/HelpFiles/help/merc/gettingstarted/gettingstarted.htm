Getting started with MERC+
 
Requirements
 

HARDWARE REQUIREMENTS - Browser or Front End input

  • PC with Internet access

  • Internet E-mail account

  • Monitor capable of displaying 1024 x 768

SOFTWARE REQUIREMENTS - Browser input

  • Browser Internet Explorer V.5.1 or higher
     It is HIGHLY recommended to upgrade to latest browser version.
    Internet Explorer V.5.0 and lower will not work satisfactory.
    Netscape cannot be used.

NOTE: If vendor chooses to input estimates via his own system - accommodations must be made in his system to generate estimates either via email attachment or web services interface

DIGITAL CERTIFICATE

Digital Certificate is needed before you can access MERC+.
It must be downloaded and installed in the computer. The certificate contains all user specific information about the respective authorization limits of the individual user. Each time a user enters MERC+, the system will automatically check if the Digital Certificate is available in the computer before granting access.

See how to get Digital Certificate here
