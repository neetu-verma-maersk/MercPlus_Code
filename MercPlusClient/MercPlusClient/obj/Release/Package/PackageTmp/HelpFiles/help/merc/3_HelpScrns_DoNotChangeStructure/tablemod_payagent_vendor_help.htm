MERC+ Pay Agent-Vendor management
 
Pay Agent-Vendor.

The Pay Agent-Vendor management screen is used to associate a Pay Agent to a Vendor.
The Vendor is the owner of a repair shop and a Vendor can own one or more shops. By setting up a Pay Agent-Vendor relationship rather than a Pay Agent-shop relationship some set up work can thus be saved when a Vendor owns more than 1 shop.

The RRIS Payment Account Code and the RRIS supplier code in MERC+ is validated in RRIS, therefore setting up account and supplier code in MERC+ requires the account and supplier code is existing in RRIS otherwise the update will not be accepted in MERC+.
The Pay Agent-Vendor association is only used when automatic transmission to RRIS or FACT takes place, if no automatic transmission of repair costs are to take place it is not necessary to set up a Pay Agent-Vendor association.

The RRIS supplier code is the equivalent to MERC+ vendor code. It may be the same in both systems but this is not necessarily the case.
 Corporate Pay Agent code, RRIS Payment Account Code and the RRIS supplier code must be obtained from local finance management.
MERC+ vendor code can be found in MERC+ under "Vendor" and Shop-Vendor association can be found "Shop profile".

Tips

Before attempting to set up a Pay Agent-Vendor association ensure that

  1. The Payagent is existing in MERC+.
  2. The Vendor code is existing in MERC+.
  3. You have a valid RRIS Payment Account Code to insert.
  4. You have a valid RRIS Supplier Code to insert.

It is extremely important point 3 and 4 are correct in MERC+ as otherwise there is risk of paying the wrong vendor or to use the wrong account code when the cost is send to RRIS.
 

How to Create a Pay Agent-Vendor association

1) From MERC+ main screen, click on link named "Pay Agent-Vendor".

2) Click "Add new" to create a new Pay Agent-Vendor association.

 

3) Select a Corporate Pay Agent Code from the dropdown

 

4) Select a Vendor Code from the dropdown

 

5) Enter RRIS Payment Account Code.
6) Enter RRIS supplier code for the Vendor.
7) Click "Add record".

 

How to Edit a Pay Agent-Vendor association.

1) From MERC+ main screen, click on link named "Pay Agent-Vendor".


2) Select Corporate Pay Agent Code from Dropdown and press "Update-Select MERC Vendor Code".


3) Select MERC+ Vendor Code from Dropdown and press "Update-Delete".


4) Edit RRIS Payment Account Code or RRIS supplier code as required and press "Update"



5) Pay Agent-Vendor association updated.

How to Delete a Pay Agent-Vendor association.

1) From MERC+ main screen, click on link named "Pay Agent-Vendor".


2) Select Corporate Pay Agent Code from Dropdown and press "Update-Select MERC Vendor Code".


3) Select MERC+ Vendor Code from Dropdown and press "Update-Delete".


4) Press "Delete"


5) Pay Agent-Vendor association Deleted.
