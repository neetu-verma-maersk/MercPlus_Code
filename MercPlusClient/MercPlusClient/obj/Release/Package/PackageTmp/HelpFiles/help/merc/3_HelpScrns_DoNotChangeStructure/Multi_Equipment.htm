How to use MERC+
Multiple entry estimate

The Multiple Equipment Entry screen is similar to single equipment entry screen in every respect except the following details.

  • This screen allows an user to enter a list of equipment having same profile / mode to be inserted at one go. The repairs for these equipments will be same
  • Spare parts cannot be used on equipments entered through this screen
