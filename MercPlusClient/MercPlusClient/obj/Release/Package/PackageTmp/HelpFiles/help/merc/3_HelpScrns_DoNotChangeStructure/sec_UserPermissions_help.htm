MERC+ user management
 
Setting user permissions

Setting user permissions is telling MERC+ what the user can see and do in MERC. Super users can grant permissions to other users who are at a level lower then they are. A Super user is thus not able to change his own or other Super users rights in MERC+, nor is he able to create new Super users.
If new Super users are to be created, GLBEMRtec@maersk.com must be contacted
For more information see the Security Matrix

Assigning permissions to a user takes place in 4 steps

  1. Select the users country (As updated in "Add new user" screen)
  2. Select the user name (As updated in "Add new user" screen)
  3. Assign the user to an authorisation group
  4. Assign the user to a shop or a location.

How to set or change user permissions

1) Select the user's country from the dropdown

 

2) Select the user from the user dropdown.

 

3) Clicking GET USER PERMISSIONS give the following screen .

 

4) Select the Authorization group from the dropdown.

 

In the screen below, we have assigned the user with Country Cluster permissions. We are now ready to assign a location to the user.

 

In the PERMISSIONS PREFIX field we enter the 1ST 2 or more characters of the field to be filtered.
(this to reduce the number of choices which are returned and reviewed for final selection)

To delete permissions

Click the MANAGE AUTHORIZATION GROUP button

On the next screen click the REMOVE USER AUTHORIZATION GROUP PERMISSIONS button.

 

 

FOR MSL – ENTER COUNTRY/CITY CODE

            US      WOULD RETURN ALL OF US

            USN   WOULD RETURN USNAV, USNOL etc.

            USNO WOULD RETUN USNOL

 

FOR SHOP, OR 3RD PARTY – ENTER SHOP CODE

20              WOULD GIVE ALL SHOPS STARTING WITH 20 (201, 20E etc.)

209          WOULD GIVE JUST SHOP 209

 

FOR VENDOR – ENTER DIGITS OF VENDOR CODE

 

FOR SYSTEM ADMINISTRATOR FILTER ON AREA

 

FOR COUNTRY – FILTER ON 2 DIGIT COUNTRY

 

FOR AREA – 15 AREAS – FILTER ON AREA NAME

 

FOR READ ONLY – FILTER ON AREA

 

FOR REGION FILTER ON REGION CODE
