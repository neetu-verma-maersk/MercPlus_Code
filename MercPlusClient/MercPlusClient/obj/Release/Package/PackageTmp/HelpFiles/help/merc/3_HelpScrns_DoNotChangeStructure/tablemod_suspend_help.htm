Manage Suspend Code Input Screen
 

OVERVIEW

This screen will be used to manage Suspend Codes. It provides users with an interface where they can delete or add new Suspend Codes.

AUTHORITY 

Only Administrator (System Admin, Prime Mover), CPH, area, Country, Region, and MSL (Location / Local EMR) will have access to the Suspend Code Screens.


QUERY / DELETE  








 

In order to delete an existing Suspend Code Users will first have to select the Shop from the Shop Code Drop Down List. After selecting the Shop user will have to press the ‘Delete – Get Manual Code’ button to get all the existing Manuals in Suspend table for the selected shop.

Note: Only those shops that are present in the Suspend Code table whether Active or not for which the user has permissions


 

 

 




After Selecting the Manual Code, user will have to press the ‘Delete – Get Mode’ button to get all the existing Modes in Suspend table for the selected Shop/Manual

 





 

 

After Selecting the Mode, user will have to press the ‘Delete – Get Repair Code’ button to get all the existing Repair Codes in Suspend table for the selected Shop/Manual/Mode.

 

 

 

 

After Selecting the Repair Code, user will have to press ‘Get Suspend Record to Delete’ to go to the details of the Suspend Code record.

 

 

 




 

 

From this Screen users will be able to delete existing Suspend Codes.

ADD NEW 

 

 

 

 

In order to add a new Suspend Code Users will first have to select the Shop from the Shop Code Drop Down List. After selecting the Shop user will have to press the ‘Add – Get Manual Code’ button to get the active (at least one active record) Manuals from the Manual Mode Table.

Note: Only active Shops from the Shop table for which the user has permissions.









After Selecting the Manual Code, user will have to press the ‘Add – Get Mode’ button to get the existing active Modes from the Manual/Mode table for the selected Manual.

 

 





After Selecting the Mode, user will have to press the ‘Add – Get Repair Code’ button to get all the active Repair Codes for the selected Manual/Mode from the Repair Code table.



 

 

 




After Selecting the Repair Code and Suspend Category (available in the Suspend Category table), user can add the Suspend Record to Suspend Code Table by pressing the ‘Add Suspend Record’ button.
