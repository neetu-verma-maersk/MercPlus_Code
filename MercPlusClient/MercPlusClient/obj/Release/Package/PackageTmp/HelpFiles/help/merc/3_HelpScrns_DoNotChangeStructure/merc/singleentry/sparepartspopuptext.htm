SPARE PARTS

  • Repair Code – the code from the STS Manual which is corresponding to the spare part number in the "Owner supplied parts number" field. A click on + will expand the available repair fields from 20 to 45.
  • Pcs – the number of the damaged parts under same repair code.
  • Owner Supplied Parts Number – Please fill in the Original manufactures spare part number.
