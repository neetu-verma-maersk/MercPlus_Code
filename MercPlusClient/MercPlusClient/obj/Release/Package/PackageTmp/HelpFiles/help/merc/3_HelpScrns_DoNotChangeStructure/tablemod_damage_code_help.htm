OVERVIEW

This screen is used to allow the user to add, modify and delete damage codes. It provides an interface, where user can view damage code details


AUTHORITY

Only ADMIN can update / delete a Damge Code. Other users can only query them.

 

How to query an existing damage code (CPH/Area/Country/Region/MSL/Vendor) 

1) Select a damage code from dropdown and click on 'Query'. All data will be returned as indicated.

 

How to Edit / Delete an existing damage code (Admin)

1) Select a damage code from dropdown and click on 'Query'. All data will be returned as indicated.

Make changes and click submit to update
Click Delete to remove the damage code.

How to Add a new damage code (Admin)

1) Click on "Add New Damage Code".

2) Fill in all the details as shown.    

  • The 'Damage code' should be two characters
  • The 'Name' and 'Description' fields cannot be left blank
  • 'Numerical code' has be 5 digit long
  • 'Change User' and 'Change Time' is updated after 'Submit'

3) Click Submit. A message will be displayed as shown
