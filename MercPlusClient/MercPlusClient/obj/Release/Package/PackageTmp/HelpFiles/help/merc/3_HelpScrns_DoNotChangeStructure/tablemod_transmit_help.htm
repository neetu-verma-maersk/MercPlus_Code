How to add Transmit Records

1) Click ADD NEW

   

2) Fill out the following screen.
3) Click ADD RECORD.

   

 

 

To Update Transmit Records


1) Select Customer Code from the Drop Down & press 'Update - Get Mode' button
2) Select Mode from the Drop Down & press 'Update - Get Transmit Record' button
 



 Step 3 - Make changes and press 'Update' button to save the changes
